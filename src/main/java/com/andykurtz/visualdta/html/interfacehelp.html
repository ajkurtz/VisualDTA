VisualDTA Interface Help

When the program is started a coding file must be loaded by selecting the "Load" option from the "File" menu.

 

Loading a file

Select the "Load" open from the "File" menu or use the shortcut key to open the file selection window. Navigate through your folders to find a coding file, select it, and click "Open".

 

Showing the descriptions

If description text was included in the coding file, it may be shown using the "Show" button. When the descriptions are shown, the button changes to "Hide" so that the display of the descriptions may be toggled.

 

Progressive display

By using the progressive display button the conversation may be displayed one proposition at a time so that it is easier to see how the conversation flows over time. Click the "Start" button will reset the display only displaying the first proposition. Clicking "Next" will display the next proposition and clicking "Previous" will remove the last proposition. Clicking the "Stop" button will reset the display so that all the propositions are displayed.

 

Attribute Highlighting

All of the propositions that match an attribute may be highlighted. Click on a proposition to display a menu listing the attributes and the values for the selected proposition. Select one of the menu options to highlight all of the propositions that match the value for the selected attribute. Select the Clear Highlights option from the popup menu or from the Clear Highlights option of the View menu.

 

View Menu - Basic Statistics

Selecting the Basic Statistics option will show a window containing counts and percentages for the propositions, moves, and attributes. The rows in the table may be sorted by clicking on the column title.

 

View Menu - Custom Comparison

The Custom Comparison option allows two attributes to be selected and compared. The rows in the table may be sorted by clicking on the column title.

 

View Menu - Node Details

Selecting the Node Details option will open a window that will show details about the proposition that is hovered over with the mouse pointer.

 

View Menu - Clear Highlights

This will turn off all of the attribute highlights. See the section on Attribute Highlights for more information.

 

Zoom Menu - Fit To Window

The Fit To Window option will zoom the display so that all of the propositions are visible on the screen without scrolling.

 

Zoom Menu - Reset Zoom

Selecting Reset Zoom will reset the display to its default zoom level.
